# LUCUS ENGINE 2
An educational project.

## Dependencies

### WIN
- Visual Studio + Toolchain + Cmake enabled
- Cmake
- Ninja
- LLVM > 20.0

### Mac
```
brew install cmake
brew install ninja
```

### Linux
- Cmake
- Ninja
- Vulkan

## Instalation

- git clone
- cd lucusengine2
- git submodule update --init
